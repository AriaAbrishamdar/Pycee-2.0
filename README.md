### :wrench: Installation for offline side:

* Create a virtual environment to install packages
```
virtualenv venv
```
* Activate the venv
```
venv\Scripts\activate
```
* Install packages
```
pip3 install -r requirements.txt
```
* Run a test
```
python application.py test_code.py
```
